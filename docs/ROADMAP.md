# MVP Roadmap (Texas + New York)

## Launch Objective
Ship the smallest high-value intelligence product that can monetize quickly across agents, brokers, investors, and homebuyers.

## Phase 1: Foundation (Weeks 1-2)
- Finalize event taxonomy and normalized schema
- Ensure source ingestion reliability across all 10 inputs
- Enforce location normalization (TX + NY)
- Build confidence scoring and traceability to source URLs

## Phase 2: Monetizable Core (Weeks 3-5)
- Hyperlocal feed by user location preferences
- Radius-based alert rules
- Pipeline/deal/financing/distress/neighborhood signal categories
- Weekly auto-generated local summary

## Phase 3: Paid Product Readiness (Weeks 6-8)
- Persona-specific experience layers (buyer, agent, broker, investor)
- Subscription gating and tier packaging
- Core usage + conversion analytics
- Compliance QA and quality assurance workflow

## Phase 4: Growth Iteration (Post-MVP)
- Improve entity linking and dedupe precision
- Expand NY and TX submarket depth
- Add team/enterprise controls and exports
- Evaluate additional geographies using same model

## Release Readiness Checklist
- Consistent TX + NY signal quality
- Stable alert delivery
- Clear paid value by persona
- Compliance controls in place
- Revenue instrumentation enabled
