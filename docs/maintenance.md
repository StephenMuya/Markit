# Monthly Maintenance Checklist (Markit Intelligence Product)

- [ ] **Source Integrity Check**: Run scraping pipeline and verify extraction quality across all 10 sources.
- [ ] **Geo Quality Check**: Validate TX + NY location normalization and confidence labels.
- [ ] **Signal Quality Review**: Spot-check each premium signal family (pipeline, deals, financing, distress, neighborhood change).
- [ ] **Alert Reliability**: Verify radius/location-based alerting and digest generation.
- [ ] **Compliance Check**: Confirm source attribution and content-handling rules are respected.
- [ ] **Monetization Health**: Review conversion, retention, and plan usage by persona.
- [ ] **Prompt/Extraction Tuning**: Update extraction guidance when recurring classification errors appear.
